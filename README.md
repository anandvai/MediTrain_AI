# MediTrain AI Bot 🤖💊  

**Empowering Healthcare Education Through AI**  

---  

## 🌟 Overview  
MediTrain AI Bot is an innovative AI-powered platform designed to enhance medical training and education. By simulating real-time patient interactions, it helps medical professionals and students improve their diagnostic and communication skills in a user-friendly environment.  

---  

## ✨ Key Features  

- **🩺 Interactive Chatbot**: Simulates patient conversations to practice real-world scenarios.  
- **📚 Medical Training Support**: Provides tailored scenarios to refine diagnostic and interpersonal skills.  
- **💡 Health Resources**: Offers curated tips, insights, and valuable health information.  
- **🌐 Intuitive Interface**: Designed for seamless use by medical students, practitioners, and educators.  

---  

## 🛠️ Project Structure  

```plaintext  
MediTrain AI Bot/  
├── frontend/   # Streamlit application for user interaction  
└── backend/    # Flask-based API for processing user queries  

🚀 Getting Started
Frontend Setup (Streamlit)
Navigate to the frontend directory:

cd Frontend
Install the necessary dependencies:

pip install -r requirements.txt
Run the Streamlit app:

streamlit run streamlit_app.py
The frontend will be available at: http://localhost:8501

Backend Setup (Flask)
Navigate to the backend directory:

cd backend
Install the required Python packages:

pip install -r requirements.txt
Run the Flask server:

python flask_app.py
The backend will be running at: http://localhost:5000

🏆 Outcomes
📝 User Registration: Medical professionals and general users can sign up.

🧠 Technologies Used
Frontend:
Streamlit: For building an interactive and user-friendly interface.
Backend:
Flask: REST API to handle and process user queries.
LangChain: Enhances contextual AI-driven responses.
GROQ API: Empowers the AI chatbot with medical interaction capabilities.
